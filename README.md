## Comment lancer 

Pour lancer le jeu il faut d'abord copier la branche master (et non "main") sur mon github, ouvrer ensuite le projet avec l'IDE de votre choix et lancer le projet.

## Informations gameplay 

- Pour aller à droite et à gauche utiliser les flèches directionnelles.

- Pour sauter appuyer sur espace. 

- Vous pouvez tuer un ennemi en sautant dessus (attention, il faut que le sprite du joueur soit vraiment au dessus de celui de l'ennemi sinon game over).

- ne marcher pas sur les piques.
